cd = (change directory) Cambia de Directorio.
cd NombreDeDirectorio/ = nos mueve al directorio llamado.
cd .. retrocede un directorio.
cd / = nos mueve al directorio raíz.
mkdir = crea un directorio.
rmdir = elimina un directorio.
pwd = nos muestra el entorno de trabajo actual.
ls = lista los archivos y directorios dentro del directorio donde se encuentre.
ln = vinculo archivos y directorios.
sudo = (super user do) Nos da permisos y privilegios de usuarios root por defecto en /usr/bin).
dpkg = Es quien gestiona los paquetes debian en GNU/Linux. 
Tar = (Tape ARchiver) archivador de cinta es un tipo de archivo usado en entornos UNIX.
Top = Administrador de tareas (Muestra programas funcionando, rendimiento, etc).
chmod = cambia permisos de lectura, escritura y ejecución de un archivo.
chown = cambia propiedad de un archivo con respecto a un usuario.
kill = para la ejecución de un programa. 
shutdown = apaga (-h) o reinicia (-r) el ordenador en un tiempo elegido (now o cualquiera).
cp = copiar
mv = mover o cambiar nombre.
Jobs = Muestra los porcesos que esta llevando a cabo la Shell.
wget = obtiene contenido de páginas web y instala la información, soporta HTTPS, HTTP y FTP.
PING = Verifica el estado de conectividad al servidor.
sudo su = Activa super user do.
apt = Advanced Packaging Tool (Herramienta Avanzada de Empaquetado) Gestiona los paquetes de GNU/Linux.
sudo apt update = Actualiza la lista de paquetes disponibles (los repositorios de los programa).
sudo apt upgrade = Instala actualizaciones.
sudo apt autoremove = Borra dependencias no usadas.
sudo apt clean = elimina el caché de los paquetes.
sudo apt autoclean = Elimina repositorio local, la caché, y paquetes no usados.
sudo apt install = Instala paquetes.
sudo apt remove = Desinstala paquetes.
snap = equivalente de apt, instala paquetes del software desde la Snap Store.
dpkg -i <paquete.deb> = instala un paquete.
dpkg -r <paquete.deb> = Desinstala el paquete.
dpkg --configure -a = Intenta reparar el sistema, viendo programas no configurados.
